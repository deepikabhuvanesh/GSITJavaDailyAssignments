Question

Write a Java program to store the following details using appropriate data types and display them:

Student ID = 101
Grade = A
Passed = true
Age = 18

Print all values on the console.

Java Code:
==========

package dailyassignments;

public class Day1_DataTypes {

	public static void main(String[] args) {
		int studentID=101;
		char grade='A';
		boolean result=true;
		int age=18;
		
		System.out.println("Student ID = " +studentID);
		System.out.println("Grade = " +grade);
		System.out.println("Passed = " +result);
		System.out.println("Age = "+age);
	}
}
